# پروژه تحلیل فروش فروشگاه لپ‌تاپ‌سان

## 📊 درباره پروژه
این پروژه یک تحلیل کامل از داده‌های فروش یک فروشگاه خیالی در بازه ۶ ماهه است. هدف از این پروژه، استخراج الگوهای فروش، شناسایی محصولات پرفروش و ارائه پیشنهادات عملی برای بهبود عملکرد فروش است.

## 🛠️ تکنولوژی‌های استفاده شده
- Python
- Pandas (برای پردازش و تحلیل داده)
- NumPy (برای محاسبات عددی)
- Matplotlib (برای مصورسازی داده و رسم نمودار)
- Jupyter Notebook (برای اجرا و نمایش کدها)

## 📈 تحلیل‌های انجام شده
- محاسبه مجموع فروش هر محصول
- محاسبه میانگین فروش روزانه هر محصول
- شناسایی پرفروش‌ترین و کم‌فروش‌ترین محصولات
- رسم نمودار میلهای برای نمایش بصری داده‌ها
- ارائه پیشنهادات برای افزایش فروش

## 📁 ساختار پروژه
- پروژه_تحلیل_فروش.ipynb : فایل اصلی حاوی کدها و تحلیل‌ها
- داده_فروش_۶_ماه.csv : دیتاست مورد استفاده در پروژه

## 💡 نتیجه‌گیری
با تحلیل داده‌ها مشخص شد که محصول [نام محصول پرفروش] بیشترین فروش را داشته است. پیشنهاد می‌شود موجودی این محصول افزایش یافته و برای محصولات کم‌فروش، استراتژی تخفیف یا تبلیغات در نظر گرفته شود.
